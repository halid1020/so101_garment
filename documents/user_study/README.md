# User-study session packs

One printable pack per participant (P1-P5), generated from the runbook in
`documents/user_study_protocol.md`: consent/demographics checklist,
familiarisation checklist, and per-condition trial sheets, NASA-TLX, SUS
and teleoperation-feel forms in that participant's Latin-square order,
plus the closing ranking and interview sheet. The condition key printed in
each section is for the experimenter only - refer to conditions as A-D in
front of the participant.
